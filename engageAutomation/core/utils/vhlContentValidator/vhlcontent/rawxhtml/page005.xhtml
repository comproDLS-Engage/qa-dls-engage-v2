TUTORIAL

The text “Hello, My Name Is” includes these words. Match each sentence to a picture. Which words do you already know?

  1. I feel more confident when I practice before a music show.
  2. What decade is it now? In which decade were you born?
  3. Each mosaic piece has a unique shape. None are exactly alike.
  4. We anticipated that it would rain this morning, so we brought an umbrella.
  5. My brother gets annoyed when I win at chess.
  6. Compared to a rabbit, a turtle is slow.
  7. When things were hard, we persevered. We kept trying!

USE THE VOCABULARY

Talk about each image. Use the new words.

In picture A, the boy is annoyed because his sister won the chess game. He probably thought he would win!
